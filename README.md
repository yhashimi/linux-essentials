---

## Useful Commands
Command | Description :

 `pwd` | Show current directory |
 `ls` | List files and directories |
 `cd` | Change directory |
`mkdir` | Create a new directory |
 `rm` | Remove a file |
`chmod` | Change file permissions |
`sudo` | Run command as root user |
